## Recommended Image Specifications

- **Format**: JPG or WebP
- **Resolution**: 
  - Hero images: 1920x1080px (full width)
  - Card images: 800x600px or 600x400px
  - Profile images: 400x500px or 600x800px
- **Quality**: 80-90% for JPG
- **File size**: Under 500KB for web optimization

## Notes

- All images should be optimized for web use
- Use descriptive filenames
- Maintain consistent aspect ratios across similar sections
- Consider lazy loading for below-the-fold images
